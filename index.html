Création une image docker à partir de DockerFile

Préparer les fichiers
ubuser@ub-srv-2:~$ mkdir Projet_GroupeAimes
ubuser@ub-srv-2:~$ cd Projet_GroupeAimes/

Création le fichier Dockerfile cat < Dockerfile >FROM nginx:latest >ADD html/ /usr/share/nginx/html >ADD conf.d/default.conf /etc/nginx/conf.d >EOF Creation l'image docker images docker build -t registryaimes.azurecr.io/groupeaimes:1.2 . docker images Créer un conteneur docker run -d -p 80:80 --name aimes registryaimes.azurecr.io/groupeaimes:1.2 Se connecter à Azure docker login registryaimes.azurecr.io Push l'image docker push registryaimes.azurecr.io/groupeaimes:1.2
